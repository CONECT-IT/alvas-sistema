ALVAS es un CRM inmobiliario organizado por módulos de dominio, arquitectura hexagonal y lenguaje ubicuo. La documentación se divide en cuatro capas:

<div class="card-grid">
  <div class="doc-card">

    <p>DDD táctico, hexagonal, módulos, reglas de dependencia y decisiones técnicas.</p>
  </div>
  <div class="doc-card">
    <h3>Dominio</h3>
    <p>Lenguaje ubicuo: captaciones, leads, clientes, propiedades, contratos y citas.</p>
  </div>
  <div class="doc-card">
    <h3>Referencias</h3>
    <p>TypeDoc para código, Scalar para API y Storybook para UI.</p>
  </div>
  <div class="doc-card">

    <p>Procesos comerciales desde captación hasta contrato y seguimiento CRM.</p>
  </div>
</div>
