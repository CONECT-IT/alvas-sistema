---
title: Storybook
description: Documentación visual e interactiva de componentes UI de ALVAS
---

Storybook documenta los componentes visuales e interactivos de `apps/web`.

Ejecutar localmente:

```bash
bun run storybook
```

Build estático:

```bash
bun run --cwd apps/web build:storybook
```

Storybook debe cubrir componentes compartidos como sidebar/navbar, botones, inputs flotantes, formularios, paneles y estados de CRM.
