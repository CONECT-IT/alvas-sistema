ALVAS usa herramientas distintas para tipos distintos de documentación.

| Herramienta | Uso                                                   |
| ----------- | ----------------------------------------------------- |
| Starlight   | Portal principal y documentación narrativa.           |
| TypeDoc     | Referencia automática de código TypeScript.           |
| Scalar      | Referencia interactiva de API OpenAPI.                |
| Storybook   | Documentación visual e interactiva de componentes UI. |

## TypeDoc

- [Referencia API](/referencias/typedoc/api/)
- [Referencia Web](/referencias/typedoc/web/)

Generar localmente:

```bash
bun run docs:build
```
