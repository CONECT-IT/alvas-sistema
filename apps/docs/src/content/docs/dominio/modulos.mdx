## Auth

Gestiona inicio de sesión, renovación de sesión, tokens y middleware de autenticación.

## Usuarios

Gestiona usuarios, roles y datos de asesores.

## Ventas

Gestiona leads, citas, clientes y contratos desde el pipeline comercial.

## Propiedades

Gestiona inventario, propiedades captadas, propiedades de cliente y reglas de visibilidad por rol.

## Integraciones

Gestiona captaciones entrantes, principalmente WhatsApp y flujos de conversión hacia lead.

## Reportes

Resume actividad comercial y acciones registradas para lectura operativa.
