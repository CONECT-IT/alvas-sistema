description: Glosario de términos del dominio de ALVAS
---

Estos términos son parte del contrato mental del sistema.

| Término   | Significado                                                                                            |
| --------- | ------------------------------------------------------------------------------------------------------ |
| Captación | Contacto entrante todavía pendiente de revisión comercial. Puede venir de WhatsApp u otra integración. |
| Lead      | Prospecto comercial ya aceptado para gestión de ventas.                                                |
| Cliente   | Persona convertida desde un lead o registrada como cliente gestionable.                                |
| Propiedad | Bien inmobiliario administrado por ALVAS, por captación o por cliente propietario.                     |
| Cita      | Seguimiento calendarizado asociado a un lead.                                                          |
| Contrato  | Documento comercial generado cuando una oportunidad avanza a cierre.                                   |
| Asesor    | Usuario comercial que gestiona sus leads, clientes, citas y propiedades asignadas.                     |
| Admin     | Usuario operativo con visibilidad completa y gestión de usuarios.                                      |

## Regla de visibilidad

El admin ve toda la operación. El asesor ve su cartera asignada: sus leads, sus clientes y propiedades donde participa como captador o responsable.
